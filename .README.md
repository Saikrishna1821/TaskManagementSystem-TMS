# Kanban Board (React + Node.js + Supabase)

A simple Kanban board application built as a clean, production-quality assignment.

## Tech Stack

- Frontend: React (JavaScript)
- Backend: Node.js + Express (CommonJS)
- Database: Supabase (PostgreSQL)
- Auth: None
- Styling: Minimal (inline styles)

---

## Features

- Three columns: Todo, In Progress, Done
- Create tasks (title + optional description)
- Move tasks between columns
- Persist tasks using backend API
- Restore board state on page reload
